# Filtros Avanzados

Sistema de filtrado múltiple para búsqueda de compras.

## Tipos de Filtros

### 1. Búsqueda de Texto
- Busca en: factura, categoría, descripción
- Tiempo real mientras escribes

### 2. Forma de Pago
- Todos
- Efectivo
- Transferencia
- Crédito

### 3. Estado
- Todos
- Pendiente (rojo)
- Pagado (verde)

### 4. Rango de Fechas
- **Desde:** dd/mm/aa
- **Hasta:** dd/mm/aa

## Uso

1. Seleccionar proveedor
2. Aplicar filtros deseados
3. Los filtros se combinan (AND)
4. Clic en **"✕ Limpiar"** para resetear

## Características

- ✅ Filtrado en tiempo real
- ✅ Combinación de múltiples filtros
- ✅ Botón para limpiar todos los filtros
- ✅ Placeholders en campos de fecha
